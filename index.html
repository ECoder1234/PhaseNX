PhaseNX

A Gamified Browser Built For Boss Fights

PhaseNX blends an Electron browser core with boss battles, XP progression, and resource controls inspired by Opera GX. Close a tab, earn XP, and take on a new challenge.

Download! Explore Features

Latest build: v0.1.0 - Windows (64-bit)

Highlights

Phase Control Room

Fine-tune RAM caps, automatically suspend idle tabs, and mute background audio to stay laser-focused on your boss fight.

Boss Battles

Closing a tab summons a boss. Spend energy, guard, and unleash specials to claim victory and XP towards the next level.

Customizable Hub

Launchpad, history view, mini games, and avatar customization keep PhaseNX feeling personal and replayable every session.

Install Instructions

  1. Click the Download! button above to grab the latest installer.
  2. Open PhaseNX-Setup-0.1.0.exe once it finishes downloading.
  3. Follow the installer prompts. A desktop shortcut is created automatically.
  4. Launch PhaseNX and explore the Control Room in the sidebar to tweak resource limits.

Tip: If Windows SmartScreen appears, choose More info > Run anyway (PhaseNX is a hobby project without code signing yet).
